Istruzioni di installazione per Library Management System


Tutti i comandi funzionano all'interno dell'ambiente virtuale. Se lo salti, i pacchetti verranno installati a livello di sistema.

1. Creare un ambiente virtuale:
   python3 -m venv .venv
   source .venv/bin/activate   # Linux/macOS
   .venv\Scripts\activate   # Windows PowerShell

2. Aggiornare pip:
   pip install --upgrade pip

3. Installare le dipendenze di produzione:
   pip install -r requirements.txt

4. Installare le dipendenze di sviluppo:  
   pip install -r requirements_dev.txt

5. Verificare l'installazione:
   pip list
